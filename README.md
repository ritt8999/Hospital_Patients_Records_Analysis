## Hospital Patient Records Analysis

## Overview  
This project analyzes hospital patient records (2021–2024) to identify trends in patient demographics, medical conditions, length of stay, and hospital billing.

## Dataset  
- Hospital Patient Records (2021–2024)  
- 10,000+ patient records  
- Fields include age, gender, medical condition, admission & discharge dates, and bill amount.

## Tools Used  
- **SQL** – data querying and analysis
- **Python (Pandas, NumPy, Matplotlib)** – data cleaning and exploratory analysis
- **Power BI / Excel** – interactive dashboard and KPIs  

## Key Analysis  
- Patient demographics and disease distribution  
- Length of stay (LOS) analysis  
- Billing and cost trends  
- Monthly hospital admission patterns  

## Dashboard  
- KPI cards (Total Patients, Avg LOS, Total Billing)  
- Medical condition and age-group insights  
- Interactive filters for better exploration

## Outcome  
Demonstrates **SQL querying, healthcare data analysis, and dashboarding skills** using real-world hospital data.
